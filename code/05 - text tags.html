Теги для оформления текста

Strong - выделяет текст жирным начертанием. Помимо визуально эффекта, имеет смысловое (семантичское) значение. Текст выделенный таким тегом важен по смыслу, при чтении на нем стоит сделать акцент, подчеркнув его значимость.

Bold - выделяет текст жирным начертанием. Имеет только визуальную нагрузку.

Em - Emotional. Выделяет текст курсивом. Имеет смысловое значение.

I - Italic. Выделяет текст курсивом. Имеет только визуальную нагрузку.

U - underline. Тег для подчеркивания текста. Устарел в HTML5. Вместо него стоит использовать тег span с соответствующим классом.

S- Stroke. Делает текст перечеркнутым.

Sup - Верхний индекс. Отображает текст в виде верхнего индекса. Например 52 .

Sub - Нижний индекс. Отображает текст в виде нижнего индекса. Например H2O.

big - Увеличивает шрифт текста на 1, по сравнению с обычным текстом. Устарел в HTML5.

small - Уменьшает шрифт текста на 1, по сравнению с обычным текстом.

code - используется для обозначения кода в тексте. let x = 1;

q - Quote. Тег для выделения цитаты внутри абзаца. Цитата внутри текста: Lorem ipsum dolor sit amet...

blockquote - тег для выделения отдельно стоящей цитаты.
